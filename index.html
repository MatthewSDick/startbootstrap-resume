Projects

Parisi Pizza

Capstone Project: Suncoast Developers Guild

I created Parisi Pizza because I wanted to learn how to create a web site with a shopping cart. A friend who owns a restaurant mentioned a few time that he would like to have the availability for his customers to order online from his web site. I found that the basics of React would not produce what I wanted so I learned about Context and Reducer within React to make a responsive shopping cart. The them of an Italian restuarant is based on my love for Italian food. Make sure you look at the pizza creation and order page.

As I get time I would like to expand this project with the following functionality.

  • The availability for customers to create account with a Login and Password
    • Learn to hash user passwords
    • Learn to store user PCI data properly by encrypting the data at rest in the database
  • Add links to various API in order to perform credit card transactions
  • A page so the business owner can perform the following functionality
    • Create new menu categories
    • Add new menu items - with item image drag and drop for the upload
    • Change the price of an existing menu item
    • Delete menu items
